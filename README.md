# Querra - Complaint Management System

A full-stack complaint management system built using Flask and MySQL.

## 🌐 Live Demo

https://querra-complaint-management-system.onrender.com/login

## 🚀 Features

- User Registration & Login
- Admin Login
- Complaint Submission
- Complaint Tracking
- Status Updates
- Notifications
- Department & Category Management

## 🛠️ Tech Stack

- Python
- Flask
- MySQL (Aiven Cloud)
- HTML
- CSS
- JavaScript
- Render
- GitHub
